# Télécharger KlirBuild (Windows + Android)

Page publique : **https://klirline.app/download**

## Fichiers

| Plateforme | Fichier | Usage |
|---|---|---|
| Windows | `KlirBuild-setup.exe` | Installateur NSIS |
| Windows | `KlirBuild.exe` | Portable |
| Android | `KlirBuild-release.apk` | Installation directe téléphone |
| Android | `KlirBuild-release.aab` | Upload Google Play Console |

URL directes :
- https://klirline.app/downloads/KlirBuild-setup.exe
- https://klirline.app/downloads/KlirBuild.exe
- https://klirline.app/downloads/KlirBuild-release.apk
- https://klirline.app/downloads/KlirBuild-release.aab

## Rebuild CI

GitHub → Actions → **Native apps (Windows EXE + Android APK)** → Run workflow.

## Play Store

1. [Play Console](https://play.google.com/console)
2. App ID : `app.klirline.klirbuild`
3. Upload `KlirBuild-release.aab`
